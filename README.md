# My Linux dotfiles

## Configuration included

- alacritty
- bspwm
- fish
- kitty
- lazyvim
- neovim
- picom
- polybar
- sxhkd
- tmux

## Install dotfiles

```bash
./install.sh
```

## Apply starship prompt

```bash
./starship.sh
```
